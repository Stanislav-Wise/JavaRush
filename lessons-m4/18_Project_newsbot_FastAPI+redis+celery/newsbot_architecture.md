Проект Telegram-bot новостей на FatsAPI.

Стек: FastAPI, Celery, Redis, Telethon

## Функциональные блоки:
1. Сбор и парсинг новостей.
2. Очередь задач и фоновые процессы (Celery + Redis).
3. Фильтрация новостей по ключевым словам.
4. Публикация новостей в Telegram.
5. REST API на FastAPI для управления и просмотра новостей.
6. Логирование и учет истории публикаций.


## Отражение блоков в структуре проекта:
1. Сбор и парсинг новостей.
    - каталог 'app/news_parser/'
2. Очередь задач и фоновые процессы (Celery + Redis).
    - файл 'app/tasks.py' c задачами Celery
    - файл 'celery_worker.py' для запуска воркера Celery
3. Фильтрация новостей по ключевым словам.
    - 'app/'
4. Публикация новостей в Telegram.
    - каталог 'app/telegram/'
    - файл 'bot.py' для клиента Telethon
5. REST API на FastAPI для управления и просмотра новостей.
    - каталог 'app/api.py' c роутерами FastAPI
6. Логирование и учет истории публикаций.
    - базовая настройка логов в 'app/config.py'

## Архитектура и поток данных
1. Источник новостей habr, РБК
2. Наш парсер 'app/news_parser/'
3. Задачи Celery 'app/tasks.py' 


# Схема
Источник новостей -> Наш парсер -> Задачи Celery -> Redis -> Telegram
